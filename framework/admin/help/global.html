• Site Layout - set the site's layout to wide or boxed. The layout cannot be boxed while the navbar layout is vertical. When set to boxed, more options will appear to control the background:
    • Background Color - solid color for the site background
    • Site Background - custom image for the background, with full control over properties like repeat, attachment, position and size
    • Background Pattern - set one of the included patterns as the site background. Will be ignored if "Site Background" is set
  • Site Theme - the main, site-wide theme to use. Any element themes set to "Auto" will inherit this value.
  • Site Width - set the site width to a custom value, in pixels or as percentage
  • Responsive Layout - enable or disable responsive mode. If this is disabled, the site will be displayed in desktop mode on all screens.
  • Head Code - add any HTML code before the closing </head> tag of the page. Can be used, for example, to add JavaScript analytics code.
Page Loader

All the options for the page loader are self-explanatory, no need to go into more detail.

Modules
  • Icon Font Sets - select which icon font sets to load. Try to keep these to a minimum, as they can have a negative impact on load times.
  • Enable Themes - master switch for custom themes. If enabled, the "Themes" section will become available and you can manage your custom themes there.
  • Page Option Metaboxes - switch for enabling option metaboxes on pages and posts. These are used to override global options on specific pages.
  • BrowserSync Script - enable or disable the BrowserSync script for logged in admins. BrowserSync works in conjunction with the Gulp watch task to instantly update the CSS when changes are made without refreshing the page.
