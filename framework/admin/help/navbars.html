• Menu Icons - switch for displaying icons for menu items. The icons can be set on the menus management page (Appearance > Menus) for each item.
  • Menu Dropdown Arrows - switch for displaying arrows when a menu item contains nested items (submenus). Arrows will be displayed on small screens regardless of this option.
Primary Navbar
  • Padding - set the navbar's padding when it is at the top, in its normal state
  • Padding When Fixed - set the navbar's padding when it is in the fixed (sticky) state
  • Opacity - set the navbar's opacity when it is in the fixed state
  • See-Through - enabling this option will make the navbar display on top of the media header (if it is enabled), instead of being displayed above or underneath the header
  • See-Through Opacity - the opacity of the navbar when it is displayed on top of the header. 0.0 is completely transparent.
  • Position - whether the navbar should be displayed above or under the header, or overlap at the top or at the bottom if "See-Through" is enabled
Secondary Navbar

The options for the secondary navbar are self-explanatory, no need to go into any more detail.
