MIXT offers a powerful theming system for its elements, allowing you to create virtually limitless color combinations.
You can customize every aspect of the element being themed, and will intelligently fill in the gaps when a field is left empty.

Once you create your themes, you can select which one you want to use for a specific element in the options panel. For the navbar, you will also be able to select the theme per specific page.

Common Fields
  • Theme Name - the name of the theme as it will appear in the theme select input
  • Theme ID - the theme's ID. It will be used in the element's theme class and in the generated CSS rules. For example, if the ID is "aqua", the class name will be "theme-aqua".
Site-Wide Themes
  • Accent - main accent color, usually the brand color
  • Background Color - main theme background color
  • Text Color - main theme text color
  • Text Color Fade - main theme muted text color
  • Border Color - main theme border color
  • Inverse Background - color for the inverse background
  • Inverse Text Color - text color for inverse backgrounds
  • Inverse Text Fade - muted text color for inverse backgrounds
  • Inverse Border - border color for inverse backgrounds
  • Alt Background - color for the alternative background
  • Alt Text Color - text color for alternative backgrounds
  • Alt Border - border color for alternative backgrounds
  • Dark Background - checkbox for dark backgrounds. Check it if the main theme background color is dark.
Navbar Themes
  • Accent - main accent color, used for hover and active states, and links
  • Background Color - navbar background color
  • Text Color - navbar text color
  • Border Color - navbar border color
  • Menu Background - dropdown menu background color
  • Menu Text Color - dropdown menu text color
  • Menu Text Fade - dropdown menu muted text color
  • Menu Hover Bg - background color for dropdown menu items when hovered
  • Menu Hover Text - text color for dropdown menu items when hovered
  • Menu Border - dropdown menu border color
  • Dark Background - checkbox for dark backgrounds. Check it if the navbar background color is dark.
  • Enable Opacity - check to enable rgba background and borders. Regular HEX colors will still be used for browsers without rgba support
  • Inverse Text, Inverse Accent & Inverse Border - colors to use when the navbar is transparent and the background behind it is a different luminance than the navbar background color. For instance, if the navbar background color is white and the background behind it is black, the inverse colors will be used while the navbar is transparent
