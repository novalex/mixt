MIXT offers a powerful theming system for its elements, allowing you to create virtually limitless color combinations.
You can customize every aspect of the element being themed, and will intelligently fill in the gaps when a field is left empty.

Once you create your themes, you can select which one you want to use for a specific element in the options panel as well as on individual pages.

Common Fields
  • Theme Name - the name of the theme as it will appear in the theme select input
  • Theme ID - the theme's ID. It will be used in the element's theme class and in the generated CSS rules. For example, if the ID is "aqua", the class name will be "theme-aqua". If you set the ID to one already used by another theme, the first one will override the latter.
Navbar Themes
  • Background Color - color to use for the navbar background
  • Border Color - color to use for the navbar border
  • Text Color - color to use for the navbar text
  • Accent - color to use for hover and active states
  • Inverse Text & Inverse Accent - colors to use when the navbar is transparent and the background behind it is of a different luminance than the navbar background color. For instance, if the navbar background color is white and the background behind it is black, the inverse text & accent colors will be used while the navbar is transparent
  • Menu Background - color to use for the navbar dropdown menus' background
  • Menu Border - color to use for the dropdown menus' border
  • Enable RGBA - check to enable rgba background and borders. Regular HEX colors will still be used for browsers without rgba support
